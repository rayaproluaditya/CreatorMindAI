# âš¡ Installation

## 1. Clone repo

```bash
git clone https://github.com/YOUR_USERNAME/CreatorMindAI.git
```

---

## 2. Go into folder

```bash
cd CreatorMindAI
```

---

## 3. Install requirements

```bash
pip install -r requirements.txt
```

---

## 4. Install Ollama

Download from:

[https://ollama.com](https://ollama.com)

Run:

```bash
ollama run llama3
```
